# Orbix Agent

Backend NestJS para mensajeria, configuracion, suscripciones y AI.

## Requisitos

- Node.js 18+
- MySQL o MariaDB
- Prisma CLI



## Instalacion

```bash
pnpm install
```

## Migraciones Prisma

```bash
npx prisma generate
npx prisma migrate deploy
```

## Ejecutar

```bash
pnpm run start:dev
```
